# pureopus 变更日志

## 2026-02-26 06:40:00

### 实现目标
- 完成 pureopus → PhonoArk.Android 对齐计划中全部 7 个批次、22 项差异的修复

### 变更内容

#### 批次 1：音频服务抽象层 + 语音包构建（4 项）
- 新建 `AudioRepository` 单例（Hilt `@Singleton`），统一管理 TTS + MediaPlayer（WAV 播放）+ 口音切换 + 音量控制 + 停止 + 诊断
- `Accent` 枚举新增 `US_JENNY`，默认口音从 `GEN_AM` 改为 `US_JENNY`（真人语音优先）
- `ExampleWord` 和 `Phoneme` 数据模型增加 `voiceAudioPaths: Map<String, String>` 字段
- `PhonemeRepository` 在加载 JSON 时自动注入 US-Jenny WAV 路径映射
- `app/build.gradle.kts` 添加 Gradle `extractVoicePack` 任务，构建前从 `US-Jenny.zip` 解压到 `assets/US-Jenny/`
- `IpaChartViewModel`、`ExamViewModel`、`SettingsViewModel` 移除各自的 `tts` 字段，改为依赖注入 `AudioRepository`
- 播放逻辑优先级：WAV 映射 → TTS 回退

#### 批次 2：IPA Chart 功能 + 收藏重构（4 项）
- 删除独立 `FavoritesScreen` 和 `FavoritesViewModel` 的导航引用
- 底部导航从 5 项改为 4 项：IPA / Exam / History / Settings
- `IpaChartUiState` 增加 `favoriteSymbols` 集合和 `allVowelsFavorited` / `allDiphthongsFavorited` / `allConsonantsFavorited` 计算属性
- 新增 `toggleFavoriteBatch(type)` 和 `clearAllFavorites()` 方法
- IPA Chart 页面新增批量收藏按钮行（★ 元音 / 双元音 / 辅音 / 清除全部）
- 音标按钮根据 `isFavorite` 显示金色边框
- `selectPhoneme()` 末尾自动播放发音

#### 批次 3：IPA Chart 布局与交互对齐（4 项）
- 音标按钮从 `CircleShape 48dp` 改为 `RoundedCornerShape(12.dp) 68x56dp`，字号从 18sp 改为 22sp
- 各分区（元音/双元音/辅音）外加 `Card` 包裹，含分类标题
- `IpaChartUiState` 增加 `playingWordId` 标记，播放中的词有背景高亮
- 分屏布局：竖屏时上下各占一半（音标列表 + 详情面板），横屏时左右分屏（1.15:0.85 比例）

#### 批次 4：Exam 行为与显示对齐（4 项）
- 4.1 开始考试自动播放第一题（`startExam()` 末尾调用 `playCurrentPhoneme()`）— 已在之前版本实现
- 4.2 选项交互禁用等价 — 无需修改
- 4.3 **去掉考试选项中的 IPA 音标转写**（防止泄题），选项按钮只显示单词
- 4.4 播放音素按钮增加按下态反馈：加大尺寸（70% 宽 56dp 高）+ `animateColorAsState` 按下变暗

#### 批次 5：Settings 功能补齐（3 项）
- 口音选择增加 US-Jenny 选项，标注 ★ 推荐
- Settings 页面底部新增"关于"卡片：版本号 1.0.0 + GitHub 链接按钮
- `SettingsRepository` 暴露 `accentChanged: SharedFlow<Accent>`，`updateAccent()` 时广播
- `SettingsViewModel.updateAccent()` 实时同步到 `AudioRepository`
- `IpaChartViewModel` collect `audioRepository.currentAccent` Flow 实时更新

#### 批次 6：History 页面增强（2 项）
- `SessionCard` 底部增加彩色对错小点行（绿色 ✓ / 红色 ✗），直观展示各题结果
- 分页懒加载：首次加载 2 条 session，之后每次加载 3 条；`loadMore()` 方法 + "加载更多"按钮

#### 批次 7：资源与启动打磨（3 项）
- 将 PhonoArk.Android 的 `Icon.png` 拷贝到 `drawable/icon.png`，`ic_launcher.xml` 引用新图标
- 将 `logo.jpg` 拷贝到 `assets/logo.jpg`
- 创建 `SplashActivity.kt`：用 `ImageView` + `Matrix` 做全屏裁剪展示，停留 300ms 后跳转 `MainActivity`
- `AndroidManifest.xml` 将 SplashActivity 设为启动入口

#### 其他修改
- `SettingsEntity` 默认口音从 `GEN_AM` 改为 `US_JENNY`
- `AppSettingsTest` 更新为验证默认口音为 `US_JENNY`
- `strings.xml` / `strings.xml (zh)` 新增 `accent_usjenny`、`about`、`version`、`load_more` 字符串资源

### 验证结果
