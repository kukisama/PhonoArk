## 2. 核心功能审查

### 2.1 CLI 命令

| 命令 | 功能 | 验证结果 |
|------|------|----------|
| `repoops --help` | 显示帮助信息 | ✔ 通过 |

| `repoops list` | 列出所有任务及依赖关系 | ✔ 通过 |
| `repoops run <name>` | 运行指定任务（含依赖链） | ✔ 通过 |
| `repoops run-all` | 按依赖顺序运行所有任务 | ✔ 通过 |

### 2.2 任务执行引擎

- ✔ **依赖解析**：正确实现拓扑排序，先执行依赖任务再执行目标任务
- ✔ **超时控制**：通过 `CancellationTokenSource.CancelAfter` 实现，超时后自动终止进程树
- ✔ **Shell 模式**：Windows 下通过 `cmd /c` 执行，Linux 下通过 `/bin/sh -c` 执行
- ✔ **环境变量**：支持通过配置注入额外的环境变量
- ✔ **错误处理**：进程启动失败、超时、取消等场景均有明确错误信息
- ✔ **依赖失败传播**：前置任务失败时跳过后续依赖任务并报告原因
- ✔ **循环依赖检测**：通过 visited 集合防止无限递归

### 2.3 配置加载

- ✔ 支持 JSON 注释（`ReadCommentHandling.Skip`）
- ✔ 支持尾随逗号（`AllowTrailingCommas`）
- ✔ 属性名大小写不敏感（`PropertyNameCaseInsensitive`）
- ✔ 校验任务名重复、命令缺失、超时负值、依赖引用不存在、自引用等
