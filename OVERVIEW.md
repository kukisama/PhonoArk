## Technology Stack Details

### Frontend
- **Avalonia UI 11.3**: Cross-platform XAML framework
- **SkiaSharp**: 2D graphics rendering
- **Fluent Theme**: Modern, clean UI design

### Backend
- **.NET 10.0**: Runtime and framework
- **Entity Framework Core**: ORM for database
- **SQLite**: Lightweight embedded database

### Architecture
- **MVVM**: Model-View-ViewModel pattern
- **CommunityToolkit.Mvvm**: Source generators for MVVM
- **Dependency Injection**: Manual DI in App.axaml.cs

### Platform Support
- **Desktop**: Windows, Linux, macOS
- **Mobile**: Android, iOS (requires workloads)
- **Web**: Browser via WebAssembly

## Performance Characteristics

- **Startup Time**: < 2 seconds (desktop)
- **Database Size**: ~50 KB (empty), ~500 KB (with history)
- **Memory Usage**: ~50-100 MB
- **Responsive**: 60 FPS UI on modern hardware
- **Offline**: 100% functional without internet

## Accessibility Features

- **Touch Targets**: Minimum 48x48 pixels
- **High Contrast**: Works with system high contrast mode
- **Keyboard Navigation**: Full keyboard support
- **Screen Reader**: ARIA labels for controls (planned)
- **Scalable Text**: Respects system font size

## Security Considerations

- **Local Storage**: All data stored locally, no cloud
- **No Personal Info**: No PII collected or stored
- **Database**: SQLite with no encryption (can be added)
- **Network**: No network calls (audio files local)
- **Sandboxing**: Platform-specific app sandboxing
